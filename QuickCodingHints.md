| Input Type                  | First Algorithms to Consider             |
| --------------------------- | ---------------------------------------- |
| Array                       | Two Pointers, Sliding Window, Prefix Sum |
| Sorted Array                | Binary Search, Two Pointers              |
| String                      | Sliding Window, HashMap, Trie            |
| Linked List                 | Fast/Slow Pointers                       |
| Tree                        | DFS, BFS                                 |
| Graph                       | BFS, DFS, Dijkstra, Union Find           |
| Matrix/Grid                 | BFS, DFS                                 |
| Intervals                   | Sorting + Greedy                         |
| Stream/Data Feed            | Heap                                     |
| Large Constraints (`10^5+`) | O(n) / O(n log n) only                   |
| Small Constraints (`<=20`)  | Backtracking / Bitmasking                |
